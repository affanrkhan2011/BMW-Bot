# BMAX: AI-powered BMW Sales Associate

BMAX is a sleek, intelligent chatbot embedded directly into the BMW website. It acts as a virtual sales associate available 24/7.

## Installation

1. Upload the \`Upload\` folder to your web server.
2. Add your Anthropic API key to \`config.js\`. You can also configure the dealership name, phone number, and styling options there.
3. Add the following script tags to your HTML page, just before the closing \`</body>\` tag:

\`\`\`html
<script src="path/to/Upload/config.js"></script>
<script src="path/to/Upload/bmax.js"></script>
\`\`\`

4. The BMAX chatbot icon will automatically appear in the bottom-right corner of your site.
